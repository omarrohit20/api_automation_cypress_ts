# API Automation with Cypress and TypeScript

This project is set up for API automation testing using Cypress in TypeScript.

## Setup

1. Install dependencies: `npm install`

## Running Tests

- Run tests in headless mode: `npm test`
- Open Cypress UI: `npm run cypress:open`

## Project Structure

- `cypress/e2e/api/tests/` - API test files
- `cypress.config.ts` - Cypress configuration
- `tsconfig.json` - TypeScript configuration

## Configuration

Update `cypress.config.ts` with your API base URL.

Replace the example baseUrl with your actual API endpoint.
